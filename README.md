# colorShare
by: Fang-FuDaGuoNanCeYe（夫大国难测也）

这些是我常用的配色。现分享之。
These are colors I usually use. Now I'd like to share these to you.

# 内容 Contents

| 配色组名称（colorGroup）  | 内容（Content）  |
|---|---|
| colorGroup1A, colorBeijingA | 基于北京地铁配色设计的配色，先从浅到深排序，再按颜色排序 |
| colorGroup1B, colorBeijingB | 基于北京地铁配色设计的配色，先按颜色排序，再从浅到深排序 |
| colorGroup2A, colorShanghaiA | 基于上海地铁配色设计的配色，先从浅到深排序，再按颜色排序 |
| colorGroup2B, colorShanghaiB | 基于上海地铁配色设计的配色，先按颜色排序，再从浅到深排序 |

# 下载和使用 Download and Use

- Origin用户：`\othFiles`文件夹里有`.oth`文件。您可以在Origin软件中导入它们。
- Matplotlib及其它用户：`\pyFiles`文件夹里有`.py`文件，里面的配色是以`#rrggbb`形式呈现的。您可以从中获取配色。

- For Origin users: `.oth` files, are placed in the `\othFiles` Folder. These files can be imported to your Origin software.
- For Matplotlib and other users: `.py` files are placed in the `\pyFiles` Folder. Colors are written in `#rrggbb` in these `.py` files.
